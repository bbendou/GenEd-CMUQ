# Audits

with [stellic-scraper](https://github.com/cs-cmuq/stellic-scraper).

- Bio audits: [major](https://academicaudit.andrew.cmu.edu/app/program/info?id=1032), [gened](https://academicaudit.andrew.cmu.edu/app/program/audits?id=1799)
- BA audits: [major](https://academicaudit.andrew.cmu.edu/app/program/info?id=1038), [gened](https://academicaudit.andrew.cmu.edu/app/program/audits?id=1800)
